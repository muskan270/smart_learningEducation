HTML Course Topics

▶ 1. HTML Introduction [5 mins]

  • What is HTML?
  • Importance of HTML in web development.
  • Basic HTML structure (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`).

▶ 2. HTML Elements and Tags [8 mins]

  • Understanding elements and tags.
  • Block-level vs. Inline elements.
  • Common tags: `<h1>` to `<h6>`, `<p>`, `<div>`, `<span>`, etc.

▶ 3. HTML Attributes [6 mins]

  • Purpose and use of attributes.
  • Common attributes: `id`, `class`, `style`, `title`, etc.
  • Global vs. specific attributes.

▶ 4. HTML Forms [10 mins]

  • Form elements: `<form>`, `<input>`, `<textarea>`, `<select>`, `<button>`.
  • Form attributes: `action`, `method`, `name`, `placeholder`.
  • Form validation basics.

▶ 5. HTML Links and Navigation [7 mins]

  • Creating hyperlinks with `<a>` tags.
  • Absolute vs. relative URLs.
  • Navigation bars and internal linking.

▶ 6. HTML Media [9 mins]

  • Embedding images with `<img>` tag.
  • Using audio (`<audio>`) and video (`<video>`) tags.
  • Embedding YouTube videos and iframes.

▶ 7. HTML Tables [12 mins]

  • Creating tables with `<table>`, `<tr>`, `<td>`, `<th>`.
  • Adding headers, footers, and captions.
  • Table attributes: `colspan`, `rowspan`, `border`.

▶ 8. HTML Lists [5 mins]

  • Ordered lists (`<ol>`) and unordered lists (`<ul>`).
  • Creating definition lists (`<dl>`, `<dt>`, `<dd>`).
  • Nesting lists for complex structures.

▶ 9. HTML Semantic Elements and Accessibility [8 mins]

  • Importance of semantic HTML: `<header>`, `<footer>`, `<main>`, `<section>`, `<article>`.
  • Role of accessibility in HTML: `aria-` attributes, `alt` text.
  • SEO-friendly HTML practices.

Description

Every single website uses HTML - even this very course landing page! It’s an absolutely fundamental tool that every web developer uses daily, and it also happens to be very easy (and even fun) to learn. HTML is the language that provides all the content on a webpage: the text, images, links, forms, videos, and more! Learning HTML is the first step in any web development journey, and this course teaches you all the HTML you need to know.

You’ll learn how to work with text elements, links, images, audio, video, tables, forms, and semantic elements. By the end of the course you’ll be an HTML master. And the best part? The whole course can be completed in a single day!
